Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree-like structure. It allows JavaScript and other programming languages to interact with and manipulate web pages dynamically.The DOM is the bridge between HTML and JavaScript,It enables modifying content, handling events, creating elements, and more, making it an essential part of web development.

How do you select an element from the DOM?

1) Selecting by ID – document.getElementById()- Selects a single element with a specific id. Returns an element or null if not found.
2) Selecting by Class – document.getElementsByClassName()- Selects multiple elements with the same class. Returns an HTMLCollection (like an array but without array methods).
3) Selecting by Tag – document.getElementsByTagName()- Selects multiple elements with the same tag. Returns an HTMLCollection.
4) Selecting with querySelector() (Single Element)- Selects the first matching element using CSS selectors. More flexible than getElementById().
5) Selecting with querySelectorAll() (Multiple Elements)- Selects all matching elements using CSS selectors. Returns a NodeList (which supports .forEach()).
6) Selecting the Entire Document – document.documentElement- Selects the whole HTML document.

What is event delegation in the context of DOM, and why it is useful?

Event delegation is a technique where a parent element handles events for multiple child elements, even if they are added dynamically. Instead of attaching event listeners to each child, you place a single event listener on the parent and use event.target to determine which child triggered the event. This improves performance and reduces memory usage, especially in dynamic content. It is commonly used for handling lists, tables, and dynamically generated elements.

How do you manipulate an element's attributes and styles using the DOM?

You can modify an element’s attributes using .setAttribute() or directly update properties like .src, .href, or .value. For styles, you can use .style.propertyName or .classList to add/remove classes dynamically. This helps in updating UI elements dynamically.
